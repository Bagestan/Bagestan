## About

Full-stack developer with over five years of combined professional and freelance experience building web applications with Angular, TypeScript, Node.js, NestJS, and PostgreSQL — plus C#, ASP.NET, and MongoDB along the way. My work has centered on modernizing legacy systems, digitalizing administrative and industrial processes, and integrating business platforms, replacing spreadsheet- and paper-based workflows with maintainable software: frontend interfaces, REST APIs, OAuth 2.0 and Microsoft Entra ID authentication, and deployment across Linux and Windows Server.

Originally from Brazil and now based in Italy, I work professionally in English, Italian, and Portuguese.
